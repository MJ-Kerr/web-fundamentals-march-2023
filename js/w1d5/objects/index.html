JavaScript
Objects

What are they even?

Objects are the second reference data type we will learn about in JavaScript (the first was arrays).

Like arrays, objects are able to hold collections of data. Here's an example:

JavaScript objects:

  1. have "spicy brackets" (curly braces) at the beginning and ends
  2. contain "key-value pairs"
  3. each pair is separated by a comma
  4. like arrays, we can mix and match data types in objects too

Objects are similar to arrays, except that instead of indices, objects have named properties. Instead of square brackets, objects have curly braces.

Perhaps the best way to start understanding objects is to compare them to objects IRL.

In real life, objects have characteristics and actions they can do.

For instance, a sweet BMX bike has a color, a wheel size, a brand, etc. These are the characteristics of a BMX bike.

Also, BMX bikes are able to do things, like jump ramps, pop wheelies, and wipe out.

In JavaScript, we call these characteristics of an object "properties" or "keys". We call the actions an object can perform "methods". A method is simply a function that belongs to an object.
