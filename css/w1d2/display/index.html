The Display Property

Some values of the display property:

  • inline
  • block
  • inline-block

display: block

  • heading tags
  • li tags
  • div tags

another block element

Block elements do not share the same line by default, they stack. You can set width and height.

display: inline-block

Best of both worlds. Elements can be side by side (if there's room), AND you can set width and height.
