The Flexbox System in CSS

The Flexbox systems is a versatile, powerful way to arrange elements in CSS. Elements are able to be arranged in one dimension - either the x-axis or the y-axis.

First Rule of Flexbox: The parent tells the children how to behave.

Only direct children of flex containers will be flex items.

Steps to Get Elements Next to Each Other:

  1. Identify in HTML file which elements must be side-by-side.
  2. Find their parent.
  3. Put display: flex on the parent.
  4. Remember - if the elements share that parent with any elements that should not be side by side, you must isolate them in a new parent.

Flex items are only as wide as their contents.

Important Flexbox Properties:

  • justify-content - for the x-axis
  • align-items - for the y-axis

Terms to remember:

  • Elements with display: flex are called "flex containers."
  • Direct children of flex containers are called "flex items."
